# 项目简介

## 核心功能
- 基于YOLOv8的光伏电池板异物检测系统
- 支持12000张数据集，包含6类异物：鸟粪、清洁、灰尘、电气损坏、物理损坏、积雪覆盖
- 提供完整源码、标注数据集、训练好的模型及训练结果
- 支持图片、视频及摄像头进行检测
- 支持选择模型和自定义置信度、IOU阈值
- 可实时显示目标位置、目标总数、置信度等信息
- 支持批量检测并直接查看所有检测结果
- 支持检测结果保存

## 适用场景
- 光伏电站维护检测
- 电池板表面异常识别
- 自动化巡检系统
- 环境影响评估

## 技术特点
- 采用Python3.9开发环境
- 集成OpenCV、PyQt5、Ultralytics、PyTorch等技术栈
- 提供PyCharm+Anaconda或VSCode+Anaconda开发环境
- 已训练完成模型，配置环境后可直接使用
- 包含完整的UI界面源文件和图标
